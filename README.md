# Data Analysis using Python & Excel

This project performs basic data analysis on a dataset using Python.

## Features
- Read data from CSV file
- Calculate average marks
- Find highest and lowest marks

## Technologies Used
- Python
- CSV (Excel format)

## How to Run
1. Run the Python file
2. Ensure data.csv is present
